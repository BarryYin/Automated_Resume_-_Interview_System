# AI辅助招聘系统

## 项目结构
```
ai-recruitment-system/
├── backend/          # Python后端
│   ├── app/
│   ├── requirements.txt
│   └── main.py
├── frontend/         # HTML+JS前端
│   ├── index.html
│   ├── interview.html
│   ├── dashboard.html
│   ├── css/
│   └── js/
└── README.md
```

## 功能模块
- AI面试系统
- 候选人信息管理
- HR管理控制台
- 数据分析统计

## 技术栈
- 后端: Python + FastAPI
- 前端: HTML + CSS + JavaScript
- 数据库: SQLite (可扩展为PostgreSQL)
