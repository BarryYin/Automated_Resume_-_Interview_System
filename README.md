# AI全自动招聘面试系统 (Automated Resume & Interview System)

## 项目简介
基于大语言模型的智能招聘面试系统，实现从简历筛选到AI面试的全自动化流程。

## 项目结构
```
ai-recruitment-system/
├── backend/          # Python后端
│   ├── app/
│   ├── requirements.txt
│   └── main.py
├── frontend/         # HTML+JS前端
│   ├── index.html
│   ├── interview.html
│   ├── dashboard.html
│   ├── css/
│   └── js/
└── README.md
```

## 核心功能
- 🤖 AI智能面试系统
- 📋 候选人信息管理
- 📊 HR管理控制台
- 📈 数据分析统计
- 📧 邮件自动化通知
- 🎯 千人千面个性化问题生成

## 技术栈
- 后端: Python + FastAPI
- 前端: HTML + CSS + JavaScript
- AI模型: 百度文心大模型 (ERNIE)
- 数据库: SQLite (可扩展为PostgreSQL)
